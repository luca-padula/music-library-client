Welcome to the Music Libary

Welcome to my Music Library app! Browse the app to discover other people's favourite albums and artists and create an account to add some of your own. Use the navigation bar to navigate all of the features of the app. If you would like to learn more about how the app was built or are interested in more of my work, see below.

The Front End

HTML 5
CSS 3
Angular
Bootstrap

The client side is built with Angular 11 using Typescript, HTML 5, Bootstrap, and some custom CSS. It interacts with the server side REST API using services as per Angular best practices, makes use of RXJS, and was deployed using Netlify.

The Back End

JavaScript
Node.js
MongoDB
JSON Web Token

The server side is a REST API built using JavaScript with Node.js and the Express framework and deployed to Heroku. It uses MongoDB for a NoSQL database and includes security measures in Express-Validator for server-side validation and JSON Web Tokens with the Passport library for authentication.
